.NET SDK Integration

Integrate LicenPro into any .NET application (WPF, WinForms, Console, ASP.NET Core) with our official client library.

Installation

Add the LicenPro package to your project using the NuGet package manager or .NET CLI.

dotnet add package LicenPro --version 2.4.0

Basic Initialization

Before validating licenses, you must initialize the SDK with your server endpoint and the product's public key.

using LicenPro;
using LicenPro.Models.Utils;

// 1. Initial configuration
var settings = new SdkSettings {{ '{' }} 
    ServerBaseEndpoint = "https://api.licenpro.tech" 
{{ '}' }};
SdkConfiguration.Initialize(settings);

// 2. Load Public Key (Copy from Dashboard > Product > Settings)
var secrets = new LicensingSecrets {{ '{' }} 
    PublicKey = "MIIBIjANBgkqh..." 
{{ '}' }};
LicenseUtils.LoadLicensingSecrets(secrets);

Validation Logic

1

Create Client

The LicenseClient handles the heavy lifting of reading files, checking RSA signatures, and server communication.

2

Call ValidateAsync

This method returns a comprehensive status result. It checks for expiry, signature validity, and server-side revocation in one call.

// Example Validation
var client = new LicenseClient(new LicenseClientOptions {{ '{' }}
    LicenseFilePath = "license.bin",
    LicenseKey = "LP-VAL-XXXX",
    ExpectedType = LicenseType.Perpetual
{{ '}' }});

var result = await client.ValidateAsync();

if (result.Status == LicenseStatus.Valid) {{ '{' }}
    // Access licensed data
    var userEmail = result.License.IssuedTo;
    Console.WriteLine($"Welcome back, {{ '{' }}userEmail{{ '}' }}!");
{{ '}' }}

The SDK includes built-in protection against clock-winding attacks and common debugger attachment detection to ensure your licensing remains secure.
