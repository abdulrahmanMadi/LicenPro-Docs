Subscription License

Subscription licenses provide time-based access to your software with recurring billing. They're ideal for SaaS models, ensuring continuous revenue while providing customers with ongoing updates and support.

What is a Subscription License?

A Subscription License grants access to your software for a specific billing period (monthly, yearly, etc.). When the subscription expires, the user must renew to continue using the software. This model provides predictable recurring revenue.

Recurring Revenue

Predictable monthly or annual income stream

Auto-Renewal

Licenses can automatically renew with payment integration

Continuous Updates

Subscribers get all updates during their subscription

Tiered Plans

Offer different subscription tiers with varying features

Subscription Billing Periods

LicenPro supports various billing periods:

Monthly

Billed every month. Lower commitment, higher per-month cost. Good for users who want flexibility.

Quarterly

Billed every 3 months. Balance between commitment and flexibility.

Annual

Billed yearly. Usually offered at a discount (e.g., 2 months free). Best for committed users.

Custom

Set any custom duration for enterprise or special agreements.

Creating a Subscription License

1
Go to Licenses → Create License

Navigate to the license creation page from your dashboard.

2
Select "Subscription" Type

Choose Subscription from the license type dropdown.

3
Set Billing Period

Choose monthly, quarterly, annual, or custom duration.

4
Configure Subscription Tier (Optional)

If you have multiple tiers (Basic, Pro, Enterprise), select the appropriate one.

5
Enter Customer Details

Add subscriber information including name, email, and company.

6
Generate License

Create the subscription license and deliver it to the customer.

SDK Integration

C# - Handling Subscription Status
using LicenPro;
using LicenPro.Models.License;

var result = await LicenseManager.LoadLicenseAsync(licensePath);

if (result.License is SubscriptionLicense subscription)
{{ '{' }}
    // Check subscription status
    var daysUntilExpiry = (subscription.ExpirationDate - DateTime.UtcNow).Days;
    
    if (daysUntilExpiry <= 0)
    {{ '{' }}
        // Subscription expired
        ShowRenewalPrompt();
    {{ '}' }}
    else if (daysUntilExpiry <= 7)
    {{ '{' }}
        // Expiring soon - show warning
        ShowExpiryWarning(daysUntilExpiry);
    {{ '}' }}
    
    // Check subscription tier for feature access
    if (subscription.Tier == "Pro" || subscription.Tier == "Enterprise")
    {{ '{' }}
        EnablePremiumFeatures();
    {{ '}' }}
{{ '}' }}
Grace Period: Consider implementing a grace period (e.g., 3-7 days) after expiration before fully disabling the software. This gives users time to renew without losing access.
