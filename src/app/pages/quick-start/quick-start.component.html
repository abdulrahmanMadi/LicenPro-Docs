1

Create Your LicenPro Account

Sign up for a free account to get started with LicenPro.

Registration Options

You can create an account using:

  • Email & Password: Traditional registration with email verification
  • Google OAuth: Quick sign-up using your Google account
1.1
Navigate to Sign Up

Go to app.licenpro.com/auth/register or click the "Sign Up" button.

1.2
Select Your Role

Choose "Sell Software" if you're a software vendor who wants to license your products.

1.3
Complete Registration

Fill in: Full Name, Email Address, Company Name (optional), Password (min 8 characters).

1.4
Verify Your Email

Check your inbox for a verification email and click the confirmation link.

Pro Tip: Use a business email address for better credibility when distributing licenses.
2

Create Your First Product

Set up a product to organize your software releases and licenses.

2.1
Navigate to Products

From your dashboard, click on "Products" in the sidebar menu, then click "Add Product".

2.2
Enter Product Details

Fill in: Product Name (e.g., "MyApp Pro"), Description, Version (e.g., "1.0.0"), Category.

2.3
Save Your Product

Click "Create Product" to save. Your product is now ready for releases and licenses.

Example Product Configuration 
Product Name: MyApp Pro
Description: Professional productivity software for teams
Version: 1.0.0
Category: Business Software
Status: Active
3

Configure RSA Keys

Set up encryption keys to secure your license files.

Important: RSA keys are required to generate downloadable license files (.bin). Without them, you can only use license keys for online validation.
3.1
Go to Product Settings

Open your product and navigate to the "Settings" tab.

3.2
Generate RSA Key Pair

Click "Generate RSA Keys" to create a new public/private key pair (2048-bit RSA).

3.3
Download Public Key

Copy or download the public key - you’ll ship it with your application so the SDK can verify license signatures.

3.4
Store Keys Securely

The private key stays on the LicenPro server (per product) and is used to sign license files. Your app only needs the public key.

Security Note: Never share your private key. The public key can be safely distributed with your application.
4

Generate Your First License

Create a license to distribute to your customers.

4.1
Navigate to Licenses

Click on "Licenses" in the sidebar, then click "Generate License".

4.2
Select License Type

Choose: Standard (single user), Trial (time-limited), Subscription (recurring), or Floating (concurrent).

4.3
Configure License Details

Fill in: License Name, Software Release, Issued To (customer email), Expiration Date.

4.4
Generate & Download

Click "Generate License". You'll receive a unique license key and downloadable .bin file.

Success! Your license is ready. Copy the license key or download the .bin file to distribute.
5

Integrate the SDK

Add license validation to your application.

5.1
Install the NuGet Packages

Add the LicenPro SDK and required dependencies to your .NET project.

.NET CLI 
dotnet add package LicenPro
dotnet add package Microsoft.Extensions.Configuration --version 8.0.0
dotnet add package Microsoft.Extensions.Configuration.Abstractions --version 8.0.0
dotnet add package Microsoft.Extensions.Configuration.Json --version 8.0.0
dotnet add package Microsoft.Extensions.Configuration.Binder --version 8.0.0
5.2
Initialize the SDK

Configure the SDK with your public key and server endpoint.

C# - Program.cs 
using LicenPro;
using LicenPro.Models.Utils;
using LicenPro.Utilities;

// Configure SDK settings
var settings = new SdkSettings
{{ '{' }}
    ServerBaseEndpoint = "https://licenpro.runasp.net/api"
{{ '}' }};

// Load your public key (from Product Settings)

var secrets = new LicensingSecrets {{ '{' }} PublicKey = publicKey {{ '}' }};
LicenseUtils.LoadLicensingSecrets(secrets);
