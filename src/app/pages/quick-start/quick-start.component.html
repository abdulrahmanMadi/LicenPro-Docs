Quick Start Guide

Get up and running with LicenPro in just 5 minutes. Protect your software with enterprise-grade licensing in three simple steps.

Estimated time: 5 minutes

This guide covers the essential workflow from account creation to first validation.

1

Create Your Account

Sign up at app.licenpro.tech. Choose "Software Vendor" to manage your products and licenses.

Verify your email address to unlock all dashboard features.
2

Define Your Product

Navigate to Products > Add Product. This acts as a container for your keys and releases.

LicenPro Dashboard Product Management Interface
3

Setup RSA Security

In your product settings, generate an RSA Key Pair. This ensures your license files are digitally signed and untamperable.

Keep your Private Key safe. You only need to distribute the Public Key with your app.
4

Integrate the SDK

Install the NuGet package and add validation logic to your startup sequence.

// Install via NuGet
// dotnet add package LicenPro

var settings = new SdkSettings {{ '{' }} 
    ServerBaseEndpoint = "https://api.licenpro.tech" 
{{ '}' }};
SdkConfiguration.Initialize(settings);

var result = await LicenseManager.LoadLicenseAsync("license.bin");
