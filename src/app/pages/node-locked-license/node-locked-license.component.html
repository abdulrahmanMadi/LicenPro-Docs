Node-Locked License

Bind your licenses to specific hardware to prevent unauthorized redistribution and sharing.

HWID Binding

Uses CPU, BIOS, and Motherboard serials.

Activation Limits

Control the number of unique devices allowed.

Clone Protection

Prevents running in unauthorized VMs.

Implementation

Node-locking works by calculating a unique Hardware ID (HWID) for the client machine. This ID is sent to the server during activation and stored as a "linked machine."

Hardware Changes

If a user renovates their PC (e.g., changes the Motherboard), their HWID will change. You can manually reset their activation from the Dashboard to allow them to re-activate on the new hardware.
