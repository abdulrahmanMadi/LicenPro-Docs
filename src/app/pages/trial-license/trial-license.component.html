Trial License

Allow potential customers to evaluate your software for a limited time before purchasing.

Time-Limited

Hard-expiry based on days or date.

Device Bound

Prevents "trial reloading" on same PC.

Analytics

Track conversion rates in real-time.

Overview

A Trial License is the best way to convert prospects into customers. LicenPro handles secure hardware matching to ensure the same machine cannot easily start multiple trials for the same product.

Auto-Revocation

Once the trial period expires, the SDK will return an Expired status, allowing you to gracefully block features and show a "Buy Now" prompt.

Implementation

// Initialize Trial Client
var options = new LicenseClientOptions {{ '{' }}
    LicenseKey = "TRIAL-XXXX",
    ExpectedType = LicenseType.Trial
{{ '}' }};

var result = await client.ValidateAsync();
if (result.Status == LicenseStatus.Expired) {{ '{' }}
    ShowPurchaseDialog();
{{ '}' }}
