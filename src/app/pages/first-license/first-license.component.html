Generate Your First License

Learn how to create license keys for your software product. Licenses are the keys that unlock your software for customers, controlling access, features, and usage limits.

Estimated time: 3-5 minutes

Understanding Licenses

A license in LicenPro is a cryptographically signed key that grants permission to use your software. Each license contains information about the user, expiration, features, and usage limits.

Cryptographically Signed

Licenses are signed with RSA keys to prevent tampering

User Information

Embed customer name, email, and company details

Expiration Control

Set perpetual or time-limited license validity

Activation Limits

Control how many devices can use the license

Prerequisites

Before generating a license, ensure you have:

✓
Created a Product

You need at least one product to generate licenses for. Learn how to create a product.

✓
Generated RSA Keys

Your product must have RSA keys configured for license signing. Learn about RSA keys.

✓
Customer Information

Have the customer's name and email ready for the license.

Step-by-Step Guide

1

Navigate to Licenses

Access the license management section

From your dashboard, click "Licenses" in the sidebar, or go to your product and click the "Licenses" tab.

2

Click "Create License"

Start the license creation process

Click the "Create License" button to open the license creation form.

3

Select Product

Choose the target product

Choose the product this license will be for from the dropdown menu.

4

Choose License Type

Select the appropriate licensing model

  • Standard: Perpetual single-user license
  • Trial: Time-limited evaluation license
  • Subscription: Recurring billing license
  • Floating: Concurrent user seats
  • Node-Locked: Hardware-bound license
5

Enter Customer Details

Fill in the licensee information

  • Customer Name: Full name of the license holder
  • Email: Customer's email address
  • Company: (Optional) Customer's organization
6

Configure License Settings

Set the license parameters

  • Expiration Date: When the license expires (or never)
  • Max Activations: Number of allowed device activations
  • Features: (Optional) Enable specific features
7

Generate License

Create the license key

Click "Create License" to generate the license key. The system will create a unique, cryptographically signed license.

8

Deliver to Customer

Distribute the license

Copy the license key or download the license file (.bin) to send to your customer. You can also email it directly from the dashboard.

License Key Format

LicenPro generates license keys in a standard format that's easy to copy and distribute:

Example License Key
XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

You can also download a license file (.bin) which contains the full license data including the digital signature.

License Files vs Keys: License keys require online validation, while license files (.bin) can be validated offline using your public key.

Best Practices

Use Accurate Customer Info: Always enter correct customer information. This helps with support and license management.
Set Reasonable Limits: Configure activation limits based on your licensing model. Too restrictive limits frustrate customers.
Keep Records: Always keep records of issued licenses. LicenPro stores all license history, but it's good practice to maintain your own records too.
