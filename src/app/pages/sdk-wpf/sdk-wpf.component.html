WPF Integration Guide

Complete guide to integrating LicenPro licensing into your WPF applications using MVVM patterns.

Installation

Package Manager Console
Install-Package LicenPro

Setup Steps

1
Override OnStartup in App.xaml.cs

Initialize SDK and check for existing license before showing windows.

2
Create LicenseActivationViewModel

ViewModel with LicenseKey, StatusMessage, IsActivating properties and ActivateCommand.

3
Create LicenseActivationWindow

XAML view with TextBox bound to LicenseKey and Button bound to ActivateCommand.

4
Implement ILicenseService

Service for dependency injection with IsLicenseValidAsync, ValidateAsync, HasFeature methods.

Best Practices

Use INotifyPropertyChanged: Implement property change notifications for proper data binding.
Dependency Injection: Register ILicenseService in your DI container for testability.
