WPF Integration

Implement secure licensing in WPF applications with async support and MVVM-friendly components.

MVVM Support

The WPF SDK is designed to work seamlessly with Dependency Injection and MVVM patterns. Use the ILicenseService for clean separation.

Usage Example

Initialize the service in your App.xaml.cs or DI container initialization.

public partial class App : Application {{ '{' }}
    protected override async void OnStartup(StartupEventArgs e) {{ '{' }}
        var settings = new SdkSettings {{ '{' }} ... {{ '}' }};
        SdkConfiguration.Initialize(settings);

        var licenseManager = new LicenseManager();
        if(!await licenseManager.CheckLicenseAsync()) {{ '{' }}
            new ActivationWindow().Show();
            return;
        {{ '}' }}
        
        base.OnStartup(e);
    {{ '}' }}
