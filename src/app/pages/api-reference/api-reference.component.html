REST API Reference

Complete documentation for the LicenPro REST API. Use these endpoints to automate license management, integrate with your backend systems, or build custom licensing workflows.

Authentication

All API requests require authentication using an API key. Include your API key in the request header:

HTTP Header
Authorization: Bearer YOUR_API_KEY
Get Your API Key: Navigate to Dashboard → API Keys to create and manage your API keys.

Base URL

Production
https://www.licenpro.tech/api

License Endpoints

POST /licenses

Create a new license

Request Body
{{ '{' }}
  "productId": "guid",
  "type": "Standard",
  "userName": "John Doe",
  "email": "john@example.com",
  "company": "Acme Inc",
  "expirationDate": "2025-12-31T23:59:59Z",
  "maxActivations": 3,
  "features": ["premium", "export"]
{{ '}' }}
Response (201 Created)
{{ '{' }}
  "id": "guid",
  "licenseKey": "XXXXX-XXXXX-XXXXX-XXXXX-XXXXX",
  "status": "Active",
  "createdAt": "2025-01-12T10:00:00Z"
{{ '}' }}
GET /licenses/{{ '{' }}id{{ '}' }}

Get license details by ID

GET /licenses/validate/{{ '{' }}key{{ '}' }}

Validate a license key

Response (200 OK)
{{ '{' }}
  "isValid": true,
  "status": "Valid",
  "license": {{ '{' }}
    "licenseKey": "XXXXX-XXXXX-XXXXX-XXXXX-XXXXX",
    "userName": "John Doe",
    "expirationDate": "2025-12-31T23:59:59Z",
    "type": "Standard",
    "features": ["premium", "export"]
  {{ '}' }}
{{ '}' }}
POST /licenses/{{ '{' }}id{{ '}' }}/revoke

Revoke a license

GET /licenses/{{ '{' }}id{{ '}' }}/download

Download license file (.bin)

Activation Endpoints

POST /activations

Activate a license on a device

Request Body
{{ '{' }}
  "licenseKey": "XXXXX-XXXXX-XXXXX-XXXXX-XXXXX",
  "hardwareId": "unique-hardware-fingerprint",
  "machineName": "DESKTOP-ABC123",
  "osVersion": "Windows 11"
{{ '}' }}
DELETE /activations/{{ '{' }}id{{ '}' }}

Deactivate a license from a device

POST /activations/heartbeat

Send heartbeat for floating licenses

Product Endpoints

GET /products

List all products

POST /products

Create a new product

GET /products/{{ '{' }}id{{ '}' }}/licenses

List all licenses for a product

Error Responses

The API uses standard HTTP status codes:

200
OK

Request successful

201
Created

Resource created successfully

400
Bad Request

Invalid request parameters

401
Unauthorized

Invalid or missing API key

404
Not Found

Resource not found

429
Too Many Requests

Rate limit exceeded
