API Reference

Build custom licensing workflows or integrations using our comprehensive REST API.

Authentication

All API requests require a Product API Key. You can find this in your Dashboard under Product > Settings.

Endpoints

GET /v1/license/validate

Perform a server-side validation of a license key.

Parameters

Name Type Description
key string The LP license key to validate.
hwid string Hardware ID for device binding.
POST /v1/license/activate

Activate a license on a new device.

Request Body

{{ '{' }}
  "licenseKey": "LP-XXXX",
  "deviceName": "PC-Workstation",
  "hwid": "9A8B7C..."
{{ '}' }}
Rate Limiting

Our API is limited to 1,000 requests per minute per Product Key. Exceeding this will return a 429 Too Many Requests status.
