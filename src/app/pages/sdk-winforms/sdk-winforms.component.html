WinForms Integration

Add licensing logic to your Windows Forms application using our specialized .NET wrappers.

Getting Started

The WinForms integration builds on the core .NET SDK but adds helper methods for common UI flows like activation dialogs.

1

Install Packages

dotnet add package LicenPro.WinForms
2

Initialize in Main()

Ensure you initialize the SDK before the first Application.Run() call.

static void Main() {{ '{' }}
    SdkConfiguration.Initialize("https://api.licenpro.tech");
    
    // Check license before showing main form
    var client = new LicenseClient("license.bin");
    var res = await client.ValidateAsync();

    if (res.IsValid) {{ '{' }}
        Application.Run(new MainForm());
    {{ '}' }} else {{ '{' }}
        Application.Run(new ActivationForm());
    {{ '}' }}
{{ '}' }}
Pre-built Dialogs

The WinForms SDK includes LicenseActivationDialog, a customizable form you can use to let users enter their keys without building your own UI.
